Bank Simulator 

#Description
This python project simulates a basic banking system. Users can create their account, deposit and withdraw money, check their balance, and view past transactions.

#Run
>Make sure python 3 is install on your computer
>Open the terminal or command prompt
>Navigate to the folder with your file
>Run this command: python "bank simulator version 1.py"

#What I've learned
>How to use the dictionary to store data
>How to create user-friendly menus that can handle invalid cases
>How to simulate real world banking logic
>How to generate random numbers

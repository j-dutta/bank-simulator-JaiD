Bank Simulator 

# Description
This python project simulates a basic banking system. Users can create their account, deposit and withdraw money, check their balance, and view past transactions.

# Run
>Make sure python 3 is installed on your computer
>Open the terminal or command prompt
>Navigate to the folder with your file
>Run this command on cmd prompt: python bank_sim_v4.py (Make sure you enter the version you downloaded)
